Pulsar Timing Arrays

Together with Rutger van Haasteren, I spent some time on thinking about the Bayesian analysis techniques for Pulsar Timing Arrays. The project benefitted from discussions with Patrick McDonald that started at CITA in the summer of 2005 before I moved to Leiden Observatory. I was also interested in using PTAs to search for Gravitational-Wave memory and for Fuzzy Dark Matter. I was a minor contributor to the PPTA team's recent evidence for the stochastic Gravitational-Wave Background.
